# CI/CD

Ez a repository egy egyszerű C nyelven írt számológép alkalmazást tartalmaz, amely a CI/CD folyamatok gyakorlati bemutatására szolgál.  
A projekt célja a GitHub Actions segítségével automatizált tesztelés, buildelés és release folyamat kialakítása.  
A labor során a continuous integration és continuous delivery lépéseit valósítjuk meg egy működő pipeline segítségével.
